TypeRight

TypeRight is a simple text editor application developed using Tkinter in Python. It provides a minimalist interface for users to type and save text.
Getting Started

To use TypeRight, follow the steps below:

    Clone the repository or download the script.

    Ensure you have Python installed on your machine.

    Run the script using the following command:

    bash

    python typeright.py

Features

    Save Text: Type your text in the textbox provided and click the "Save" button to save the content to a file named text.txt.

Usage

    Launch the application.
    Type your text in the textbox.
    Click the "Save" button to save the entered text.

File Management

    The entered text is saved in a file named text.txt in the same directory as the script.

Color Scheme

The color scheme of the application adapts to the system's color scheme. It uses a default blue color theme for consistency.
Dependencies

    TypeRight relies on the following Python libraries:
        Tkinter
        customtkinter
